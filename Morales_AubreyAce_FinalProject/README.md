# Study Time

## Project Overview
Study Time is a Python-based application that helps users manage their study sessions. It allows users to add, view, search, update, delete, sort, count, save, and load study session records.

## Features
- Add Session
- View Sessions
- Search Session
- Delete Session
- Update Session
- Sort Sessions
- Count Total Sessions
- Calculate Total Study Time
- Save Data
- Load Data

## Setup Guide
1. Download or clone this repository.
2. Open terminal in the project folder.
3. Run:

python src/main.py
